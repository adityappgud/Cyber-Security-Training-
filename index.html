Glossary of Key Terms

A quick reference for the essential terminology and abbreviations used in cybersecurity.

A

AAA

Authentication, Authorization, and Accounting.

A

Access Control

A security technique that regulates who or what can view or use resources.

A

ARP Spoofing

A type of MITM attack where an attacker sends false ARP messages to redirect network traffic.

A

ATO

Account Takeover.

B

Brute Force Attack

A trial-and-error method to guess information, such as passwords or PINs.

C

CIA Triad

Confidentiality, Integrity, and Availability. The foundational model for cybersecurity.

D

DDoS

Distributed Denial-of-Service.

E

Encryption

The process of converting data into a code to prevent unauthorized access.

F

Firewall

A network security device that filters incoming and outgoing network traffic.

F

FTC

Federal Trade Commission.

I

IAM

Identity and Access Management.

I

ICS

Industrial Control Systems.

M

Malware

Malicious software designed to cause damage to a system.

M

MFA

Multi-Factor Authentication.

M

MITM

Man-in-the-Middle.

N

NHS

National Health Service (UK).

O

OT

Operational Technology.

P

PAM

Privileged Access Management.

P

Penetration Testing

A simulated cyberattack to evaluate a system's security.

P

Phishing

A social engineering attack to trick users into revealing sensitive information.

P

PII

Personally Identifiable Information.

P

Privilege Escalation

An attack where an attacker gains higher-level permissions.

R

Ransomware

A type of malware that encrypts files and demands a ransom.

S

SCADA

Supervisory Control and Data Acquisition.

S

SIEM

Security Information and Event Management.

V

VPN

Virtual Private Network.

Z

Zero Trust

A security model that trusts no user or device by default.
